# Космический Телеграм

Проект для автоматизации сбора фотографий космоса и их публикации в Telegram-канале.

## Как установить

### Предварительные требования

- Python 3.6 или выше.
- Установленный менеджер пакетов pip.

### Установка зависимостей

1. Склонируйте репозиторий и перейдите в папку проекта:
   git clone <URL вашего репозитория>
   cd <название папки проекта>

2. Создайте файл requirements.txt:
   echo "python-dotenv>=1.0.0
   requests>=2.31.0
   python-telegram-bot>=20.0" > requirements.txt

3. Установите зависимости:
   pip install -r requirements.txt

4. Проверьте, что все библиотеки установились без ошибок:
   pip list

## Настройка API-ключей

Для работы скриптов потребуется два ключа. Их необходимо сохранить в переменные окружения.

### 1. NASA API Key

Этот ключ нужен для загрузки фотографий с серверов NASA.

1. Перейдите на сайт NASA API: https://api.nasa.gov/
2. Нажмите Generate API Key.
3. Заполните форму (укажите имя и email) и нажмите Signup.
4. Скопируйте полученный ключ.

### 2. Telegram Bot Token и Chat ID

Требуется для автоматической отправки изображений в ваш Telegram-канал.

1. Откройте Telegram и найдите бота @BotFather.
2. Отправьте команду /start.
3. Создайте нового бота командой /newbot.
4. Следуйте инструкциям: укажите имя и username для бота (username должен заканчиваться на bot).
5. После создания вы получите Bot Token — сохраните его.
6. Добавьте созданного бота в ваш Telegram-канал в качестве администратора (права на публикацию сообщений обязательны).
7. Получите Chat ID канала:
   - Перешлите любое сообщение из вашего канала боту @userinfobot.
   - Или воспользуйтесь API-запросом (замените BOT_TOKEN на ваш): https://api.telegram.org/botBOT_TOKEN/getUpdates
   - Chat ID канала обычно начинается с минуса (например, -1001234567890).

### Проверка ключей

Убедитесь, что:
- Ключ NASA содержит 40 символов (буквы и цифры).
- Токен Telegram имеет формат 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz.
- Chat ID — число, для канала начинается с минуса.

## Настройка переменных окружения

Для безопасного хранения ключей их нужно поместить в переменные окружения. Создайте в корне проекта файл .env:

NASA_API_KEY=ваш_ключ_от_NASA
TG_BOT_TOKEN=ваш_токен_от_BotFather
TG_CHAT_ID=ваш_chat_id_канала

Примечание: убедитесь, что файл .env добавлен в .gitignore, чтобы случайно не опубликовать секретные данные на GitHub.

### Проверка переменных окружения

Запустите в консоли (временно, для проверки):
python -c "from dotenv import load_dotenv; import os; load_dotenv(); print('NASA_KEY:', bool(os.getenv('NASA_API_KEY'))); print('TG_TOKEN:', bool(os.getenv('TG_BOT_TOKEN'))); print('CHAT_ID:', bool(os.getenv('TG_CHAT_ID')))"

Вывод должен быть:
NASA_KEY: True
TG_TOKEN: True
CHAT_ID: True

## Запуск

Для запуска бота выполните команду:
python main.py

### Проверка работоспособности

После запуска бот должен:
1. Загрузить фотографии с серверов NASA.
2. Отправить их в указанный Telegram-канал.
3. Вывести в консоль сообщение об успешной отправке.

Если вы видите ошибку — сверьте ключи и права бота в канале.
