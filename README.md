# Космический Телеграм

Данный проект поможет автоматизировать сбор фотографий космоса и автоматически публиковать их в телеграмм канал

## Как установить

### Настройка API ключей

Для корректной работы проекта вам потребуется получить два API-ключа:

### 1. NASA API Key (для загрузки изображений)

Проект использует API NASA для получения фотографий.

Получить ключ можно здесь:
https://api.nasa.gov/

Шаги:
Перейдите на сайт NASA API.
Нажмите "Generate API Key".
Заполните форму (имя, email).
После этого вы получите API-ключ.

### 2. Telegram Bot Token (для публикации в канал)

Чтобы отправлять изображения в Telegram-канал, необходимо создать бота.

Сделать это можно через Telegram-бота @BotFather

Шаги:
Откройте Telegram и найдите @BotFather.

Отправьте команду:

/start

Затем:

/newbot
Укажите имя и username для бота.
После создания вы получите Bot Token — сохраните его.

### Получение Chat ID канала

Чтобы бот мог публиковать сообщения:

Добавьте бота в ваш канал как администратора.
Получите chat_id:
Можно использовать бота @userinfobot
Или отправить сообщение в канал и получить ID через API

Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```
